BELOTE TIME

Jeu de belote sur plateforme Android Smartphone et Tablettes

Un projet étudiant

Ce projet est réalisé dans le cadre du module Projet du Master 1 Informatique de l’université Toulouse III Paul Sabatier. Ce projet devra être finalisé dans le temps imparti suivant : début janvier jusqu’à fin avril 2014.

Description du projet

Notre projet consiste à réaliser un jeu de belote sous plateforme Android. La belote est un jeu de cartes qui se joue à 4 par équipes de 2 joueurs. Assez proche du tarot, ce jeu de cartes – qui mélange hasard et stratégie – est réputé pour sa convivialité. L’idée de développer un jeu de belote a été dans un premier temps un projet personnel, que nous avons ensuite décidé de proposer dans le cadre de l’UE Projet. Le choix de développer pour la plateforme Android nous est apparu plus pertinent, celle-ci étant aujourd’hui une référence dans le domaine (Android dispose de 65% de part du marché ).
